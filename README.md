# EbonholdStuff

EbonholdStuff is a Quality of Life AddOn made **for Ebonhold**.

## What it does
- Automate vendoring entire inventory.
- Allows blacklisting to disable vendoring certain items.
- Deletion of non-vendorable items. Configured list only, does not delete anything by default.
- Has settings so you can turn features on/off

## Installation
1. Download the latest release (or clone this repo).
2. Put the folder here:
   `Ebonhold/Interface/AddOns/`
3. Make sure the folder is named:
   `EbonholdStuff`
4. Start the game and enable the AddOn.

## Settings
Configure it in-game. If something feels unsafe, turn it off..
Use **/ehs** to open Interface Settings or use Game Menu.

## Reporting bugs
If something breaks, include:
- What you were doing
- The full Lua error text
- Any relevant settings you had enabled
